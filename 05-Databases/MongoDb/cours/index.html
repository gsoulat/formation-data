Formation MongoDB

Cours Modulaire - De Zero a Hero en Data Engineering

7 Modules | 1 heure

Bienvenue dans la formation MongoDB !

Cette formation est divisee en 7 modules progressifs pour vous permettre de maitriser MongoDB, de l'installation aux cas d'usage avances en Data Engineering.

Programme de la formation

Module 1: Introduction a MongoDB

Decouvrez MongoDB, les bases NoSQL, les differences avec SQL et les concepts cles (documents, collections, BSON).

10 min

Module 2: Installation et Configuration

Installez MongoDB avec Docker, docker-compose, configurez mongosh et connectez-vous avec Python PyMongo.

8 min

Module 3: Operations CRUD

Maitrisez les operations Create, Read, Update, Delete avec des exemples MongoDB shell et Python.

10 min

Module 4: Framework d'Agregation

Utilisez $match, $group, $project, $lookup, $unwind pour des analyses de donnees complexes.

10 min

Module 5: Index et Performance

Optimisez vos requetes avec les index (simples, composes, texte, geo), utilisez explain() et apprenez les bonnes pratiques.

8 min

Module 6: Modelisation des Donnees

Apprenez a modeliser vos donnees: embedded vs referenced, patterns courants, regles d'or.

8 min

Module 7: MongoDB en Data Engineering

Integrez MongoDB dans vos pipelines: ETL avec Python, Change Streams, transactions ACID, backups.

10 min

A la fin de cette formation, vous saurez :

  • Installer et configurer MongoDB (local et cloud)
  • Effectuer toutes les operations CRUD
  • Creer des pipelines d'agregation complexes
  • Optimiser les performances avec les index
  • Modeliser efficacement vos donnees
  • Integrer MongoDB dans des workflows Data Engineering
